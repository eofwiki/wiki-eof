# Talent System

**Talent System** allows players to customize and enhance their character by unlocking powerful abilities through a  skill tree. This system is designed to support a wide range of playstyles by offering a mix of strength, agility & intelligence skills.\
\
**Unlocking Abilities**\
Abilities within the skill tree are unlocked using a special resource called Medusal Cells.

**Abilities**

<summary>Active Abilities</summary>

Active ability is a gameplay action which requires player‚Äôs input to be initiated. All abilities have their unique effects, cooldowns and deplete energy on use. Each active ability slot has a keybind, default being \[RMB], \[Q], \[E] & \[Space].

* \[RMB], \[Q] and \[E] - active ability slots with player‚Äôs chosen abilities to be assigned into
* \[Space] - activates ‚ÄúDash‚Äù ability. This is default active ability, provided from the start and cannot be replaced.

**Acquisition**

Active abilities can be acquired by activating the skills in the skill tree by using the surgery table hideout module.

**Assignation**

Whenever skill with an active ability is activated in the skill tree, it provides access to its respective ability which can then be assigned into any active ability bar slot. Same ability can only be assigned into 1 slot. Assigned skills‚Äô borders are visually distinct from the unassigned skills.

</details>

<details>

<summary>Passive Abilities</summary>

Most of the skills provide static character parameters like attribute and stat increases while the minority provide more interesting, conditional and situational passive abilities. Passive abilities are instant or temporary combat effects which trigger or activate under certain conditions. They do not have any cooldown and do not deplete energy, unless being a part of its effect.

</details>

**Skill tree**

<details>

<summary>Skill Nodes</summary>

Skill node is a part of the skill tree which can be chosen and learned. Once learned, it grants one of the following:

* Static Attribute Bonus
* Passive Ability (Unique)
* Active Ability (Unique)

</details>

<details>

<summary>Structure</summary>

The tree consists of a spectrum of skills spread out in 3 different axis according to Character‚Äôs Primary Attributes:

* Strength
* Agility
* Intelligence

**Strength** axis skills are focused on pure damage dealing and survivability.

**Agility** axis skills are focused on mobility and intricate attacks.

**Intelligence** axis skills are focused on disabling/weaking enemies and resistances to DMG elements.

Player can choose skills in between axis - resulting in mixed approach in between axis skills.

Skills that are closer to the center provide weaker and universal balances.

üí° Example: Damage increased by 10%

Skills that are further down any direction are very powerful, but very situational or provide a weakness in some form in addition to its main effect.

üí° Example: While not moving, you deal 20% more damage, but receive 10% more damage.



</details>

<details>

<summary>Rules &#x26; Progression</summary>

Player starts at the skill node in the middle of the skill tree. In order to activate a skill node, player must use the surgery table hideout module, find a skill node which connects to the starting node or any other node that is already active, make sure that it has an empty slot and place the mutagen into it. Once activated, player is immediately granted with that node‚Äôs effect and unlocks additional pathways of nodes.

‚ÑπÔ∏è Mutagen is a purified form of ‚Äúalien material‚Äù found in raids.

Player can also deactivate skills one by one in such a manner that after deactivating a skill the skill requirements for the rest of the skills are still met (pathways). No refunds are given for deactivating a skill.

**Skill Tree Reset**

During game updates which include skill tree pathway changes or adds additional rules or skills which in some way might break the current skill tree, the whole skill tree is reset and player is refunded with all materials that were used for activated skills in the old version of the skill tree.

</details>
